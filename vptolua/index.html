VP2LUA - Virtual Piano to Lua Converter

VP2LUA is a tool created by hellohellohell012321 that converts Virtual Piano sheet music into Lua code. This code can be executed in an executor, allowing you to autoplay songs. It is a branch off of the TALENTLESS GUI, a piano autoplayer which uses the same method of fake-playing the virtual piano.

Important Notes:

  • Some songs with funky rhythms (triplets, 0.75 beat rests) may not be detected properly when converting the MIDI file into virtual piano sheets.
  • If this happens, try putting your MIDI into half-time mode to slow notes by 2x but remember that it likely will not fix the issue.
  • Remember to double the BPM in the final product if you use half-time mode.
